What is a prototype?

An early model of something that can be replicated and improved upon in successive versions.

A prototype is a basic version of a product, often created to test a particular aspect of that product.
(Example of an IRL prototype)

Why should you prototype?

Discover problems early

A prototype allows you to test with real customers

  • Concepts / features
  • Interactions and user flows
Early concept testing allows you to validate features and high-level concepts.
